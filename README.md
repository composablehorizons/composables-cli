# Composables CLI

A CLI for creating Compose Multiplatform apps with [Composables UI](https://composables.com/ui).

![Demo](https://github.com/user-attachments/assets/c60cfcf9-1fec-4364-953f-1a8a2ce3d9c6)

## Quickstart

Run the following command to install Composables CLI: 

```shell
npm install -g composables-cli
```
