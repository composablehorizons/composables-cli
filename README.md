# Composables CLI

A CLI for creating Compose Multiplatform apps with Composables UI.

![Demo](https://github.com/user-attachments/assets/c60cfcf9-1fec-4364-953f-1a8a2ce3d9c6)


## Installation

```shell
npm install -g composables-cli
```

## Quick Usage

```shell
composables init composeApp
cd composeApp
./gradlew run
```
